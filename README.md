**注意事项：**

1）农历闰月，需在数字前面输入“闰”字，比如“闰3”。

2）生于“子时”的人，以现代的日期时间习惯来使用即可。

比如：3日23:30，即为3日的夜子时；4日00:30，即为4日的早子时
